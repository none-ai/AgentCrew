# OpenAgent Examples

This directory contains various examples demonstrating OpenAgent's capabilities.

## Running Examples

Make sure OpenAgent is installed:

```bash
pip install AgentCrew
```

Then run any example:

```bash
python examples/basic_usage.py
python examples/task_execution.py
python examples/messaging.py
python examples/team_management.py
```

## Examples Overview

| Example | Description |
|---------|-------------|
| `basic_usage.py` | Load teams, list agents, get status |
| `task_execution.py` | Create, assign, and execute tasks |
| `messaging.py` | Inter-agent messaging and notifications |
| `team_management.py` | Create custom teams, manage agents |
